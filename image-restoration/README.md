# image-restoration

Projet dédié à l'exploration de techniques de restauration d’images dégradées, dans le cadre d’un mini-mémoire.

## Architecture

- `Mini-mémoireM1/`
  - Dossier principal du mini-mémoire avec sous-dossier `Book/` (peut contenir le rapport, des ressources ou du code)
  - Organisation probable en notebooks/jupyters, scripts Python, ressources ou jeux de données illustrant différentes approches de restauration

## Technologies

- Python (langage principal pour la rétro-ingénierie et l’expérimentation)
- Probable recours à NumPy, scikit-image ou d’autres packages scientifiques pour la manipulation d’images et la restitution expérimentale
