# computer-vision

Projet académique sur la vision par ordinateur, expliquant et appliquant différentes méthodes de traitement et d'analyse d'images.

## Architecture

- `Vision&Par&Ordinateur/`
  - Dossier principal regroupant les modules et ressources du projet
  - `transformation ponctuelle/` : contient des scripts ou notebooks liés aux transformations ponctuelles appliquées sur les images

## Technologies

- Python (langage principal pour les traitements d’images et l’expérimentation)
- Utilisation probable de bibliothèques classiques telles que NumPy, OpenCV ou Matplotlib pour le traitement et la visualisation (selon les standards en vision par ordinateur en Python)
